Project Góry

Sobre

Este projeto foi desenvolvido para o Instituto Federal de Educação da Paraíba [IFPB] - Campus Cajazeiras como projeto final referente à disciplina de Banco de Dados Não-Convencionais ministrada pelo professor Paulo Freitas.

A origem do nome Góry é de origem polonesa que significa Montanha, onde todos os conteúdos formariam a tal montanha.

O projeto consiste em desenvolver um sistema de gerenciamento de conteúdos. Um CMS (Content Management System) é um sistema utilizado para criar, classificar e publicar conteúdo digital. Cujo objetivo principal desse trabalho é trabalhar com os bancos NoSQL e implementar o conceito de persistência poliglota. As únicas restrições aos bancos utilizados é que eles devem ser gratuitos, de código aberto e de modelos de dados diferentes.

O domínio do CMS a ser desenvolvido é livre, mas o sistema deve ser WEB. Também não há restrições quanto às linguagens de programação utilizadas no desenvolvimento.
